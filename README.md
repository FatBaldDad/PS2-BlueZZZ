# PS2-BlueZZZ

**PS2-BlueZZZ** is an **FBD / Fat Bald Dad** project focused on internal Bluetooth audio integration for PlayStation 2 consoles.

The goal of this project is to explore cleaner ways to add Bluetooth audio capability to compatible PS2 systems while keeping the install compact, serviceable, and better suited for custom internal builds.

PS2-BlueZZZ is being developed as a possible future FBD kit, product, service-install option, or documented PlayStation 2 modding platform.

> [!IMPORTANT]
> PS2-BlueZZZ is currently in private development.
>
> This public repository exists to document the project name, public goals, development status, and ownership of the FBD PS2-BlueZZZ project.
>
> This is **not** an open-source hardware or manufacturing release at this time.
>
> Detailed PCB files, schematics, Gerber files, manufacturing files, wiring diagrams, installation methods, test data, revision history, board renders, prototype photos, and internal development notes are intentionally private until the project is further tested and ready for release.

---

## Project Purpose

Bluetooth audio can be useful for PlayStation 2 builds where the user wants a cleaner setup, fewer external adapters, or a more modern audio option for late-night play.

Many Bluetooth audio solutions are external, bulky, or not designed specifically around internal PS2 installation. PS2-BlueZZZ is intended to explore a cleaner internal approach that fits better with custom PS2 builds and future FBD mod-management ideas.

The long-term goal is to create a controlled and serviceable way to add Bluetooth audio functionality while keeping the console layout clean and easy to support.

---

## What PS2-BlueZZZ Is Trying to Do

PS2-BlueZZZ is intended to explore:

- Internal Bluetooth audio support for compatible PS2 builds
- Cleaner integration than external Bluetooth adapters
- Service-friendly installation planning
- Better control over pairing, enable/disable behavior, and user interaction
- Compatibility with custom PS2 aesthetic builds
- Future integration with larger FBD mod-management projects
- A possible future kit, installation service, or documented hardware platform

The public repository will grow as the project matures, but the working development files are kept private for now.
