# Contributing

Thank you for your interest in PS2-BlueZZZ.

This public repository is limited to announcement, ownership, and high-level documentation purposes while the project remains in private development.

## Public contributions that are welcome

Contributions are currently limited to:

- Public feedback
- Compatibility observations
- Documentation corrections
- Research references
- General feature suggestions

## Not currently accepted

This repository is not accepting:

- Hardware design pull requests
- PCB layout pull requests
- Fork-based development
- Clone hardware designs
- Manufacturing files
- Installation method submissions
- Productized derivatives
- Alternative implementations based on private FBD work

## Submission guidance

If you open an issue or submit a documentation-focused pull request, please keep it high level, accurate, and free of private implementation details.

Do not submit schematics, PCB files, Gerbers, KiCad projects, manufacturing documentation, detailed wiring information, internal test data, or private development notes.
